Parameters

deviceAddress
Type: SystemString
Use port name (example: "COM1") for serial connection, IP Address (example: "192.168.1.1") for TCP/IP connection, or empty string ("") or ("CAN") for CAN Open connection. This input parameter is case insensitive

Return Value

Type: Boolean
true - no error, false - error occurred ErrorCode or ErrorMessage.
Examples
This sample shows how to call the Connect(String) method and how to use IsConnected Property to validate that the connection was successful
Copy
/// class Program
{
    static int Main() 
    {
        MotionController controller = new MotionController(ControllerType.AG300);
        if(controller.Connect(COM1)) {
            System.Console.WriteLine("Connect Success")
        }
        else 
        {
            System.Console.WriteLine($"Connect Failed: {ErrorMessage}")
        }
        Console.WriteLine(controller.IsConnected ? "Connection, Validated" : "Connection is not Connected, validated not successful");
    }
}
As can be seen above, this method returns a true or false statement, a return of false means that the method call was unsuccessful. The reason for this failed method call is described in more detail in the ErrorMessage property
