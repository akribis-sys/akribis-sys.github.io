MotionController Class

High Level API Documentation
This is the entry class of HLAPI to access AAMotion Controllers, There are three basic components that MotionController motion controller may have; ISingleAxis, IIOModule and IGroupAxis. These interfaces act as the pathway to access these components. They can all be consider a Node, INode
