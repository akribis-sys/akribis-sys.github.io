Home
Home the axis motor (blocking). The method will block until either the motion completes, an error occurs, or times out.
Jog
Jog the axis motor using the velocity and direction provided (non-blocking). This command does NOT wait for the motion to stop. Use the Stop method to stop the motor jogging.
MoveAbsolute
Move the axis motor to an absolute position (non-blocking). This command does NOT wait for the motion to complete before returning. If you want to wait for the motion to complete, please use the WaitMotionDone method after this command.
MoveRelative
Move the axis motor relative to its current position (non-blocking). This command does NOT wait for the motion to complete before returning. If you want to wait for the motion to complete, please use the WaitMotionDone method after this command.
