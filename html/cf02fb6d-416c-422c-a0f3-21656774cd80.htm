Setup

      "CommServerVersion"="1.0.0.1"

Ethernet

This section denotes common things that the user can refer to when troubleshooting issues relating to Ethernet Communication

  • Note that when it comes to the Ethernet Physical layer, AG300 and Central-I CIG-MAS01 master controller, only support up to 10/100MB/s data rate

  • If you are attempting to connect to AAMotion using TCP/IP, then make sure you set the Ethernet connect to static IP. The default IP address of the controller is "172.1.1.101". Alternatively to check what is the current IP address settings, connect either using CAN bus or RS232. Then using PCSuite, query the values AEthernetIP[1]/[2]/[3]/[4]. This setting will be the IP address of your controller. If you have changed the IP address, then make sure you power cycle the controller or type AReset in the PCSuite Terminal to do a soft reset.

    • To verify that you are able to talk to the controller without going through the PCSuite, open up command prompt and type in "Ping xxx.xxx.xxx.xxx" to make sure the controller is replying to you.

Bug Report

When a bug has been found, it is important to report it so that the technical team can be aware of its existence. Thereafter this bug will be systematically tracked and fixed

In order to assist with the initial confirmation and/or diagnosis of the bug, the user can help by sending along a log traces which is generated transparently and will reside in the fixed location of C:\PCSuite\Logs\Diagnostics*.log
