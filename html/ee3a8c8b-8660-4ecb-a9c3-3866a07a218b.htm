High Level API Documentation

The topics in this section describe the various changes made to the HLAPI AAMotion over the life of the project.

The versioning format of the AAMotion HLAPI follows Semantic Versioning(SemVer) where by X.Y.Z.A denotes Major, Minor, Patch, Build. Therefore any user of AAMotion HLAPI can be assured that this DLL follows the rules dictated by Semantic Versioning when it comes to incrementing of version number
