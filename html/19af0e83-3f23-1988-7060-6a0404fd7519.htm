High Level API Documentation

The MotionController type exposes the following members.

Properties
  NameDescription
CommmunicationType
This property returns the current communication type CommmunicationType that is set for the controller This property is only set when Connect(String) method is attempted with an corresponding string input argument as defined in Connect(String)
ErrorCode
This property returns the error code of the last error incurred by MotionController This message is reset after a Disconnect or Connect(String) method has been called ErrorCode is the numerical equivalent of ErrorMessage
ErrorMessage
This property returns the error message of the last error incurred by MotionController This message is reset after a Disconnect or Connect(String) method has been called ErrorMessage is the numerical equivalent of ErrorCode
IsConnected
This Property is will indicate whether the instantiated class MotionController is currently in a live session with communicator and is currently connected with a controller with a connection type of CommmunicationType
Type
The controller can only be set when MotionController class is instantiated MotionController
