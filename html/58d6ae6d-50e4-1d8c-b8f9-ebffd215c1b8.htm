Methods
  NameDescription
ConfigurePDO
This method is to configure SingleAxisPropertyList PDO settings. For detailed explanation on PDO see INode

This method configures SingleAxisPropertyList SDO settings For detailed explanation on SDO see INode
GroupEnable
Home
Home the axis motor (blocking). The method will block until either the motion completes, an error occurs, or times out.
Jog
Jog the axis motor using the velocity and direction provided (non-blocking). This command does NOT wait for the motion to stop. Use the Stop method to stop the motor jogging.
MoveAbsolute
Move the axis motor to an absolute position (non-blocking). This command does NOT wait for the motion to complete before returning. If you want to wait for the motion to complete, please use the WaitMotionDone method after this command.
MoveRelative
Move the axis motor relative to its current position (non-blocking). This command does NOT wait for the motion to complete before returning. If you want to wait for the motion to complete, please use the WaitMotionDone method after this command.
ServoOff
Turn the servo motor off.
ServoOn
Turn the servo motor on.
SetPositionsZero
Stop(AxisStopType)
Stop the current motion.
Stop(AxisStopType, Boolean)
Stop the current motion and turn off the motor.
WaitCommmutationDone
Wait for commutation to complete (blocking). The method will block until either commutation completes, an error occurs, or times out.
WaitHomingDone
Wait for the current home command to complete (blocking). The method will block until either homing completes, an error occurs, or times out.
WaitMotionDone
Wait for the current motion to complete (blocking). The method will block until either the motion completes, an error occurs, or times out.
