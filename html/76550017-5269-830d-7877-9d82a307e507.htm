The ISingleAxis type exposes the following members.

Properties
  NameDescription
ActualPosition
The actual position that is being reported by controller
ActualVelocity
The actual velocity is the value of the actual velocity that the controller is measuring
Axis
(Inherited from INode.)
CommandPosition
The command position is the instantaneous value of the position at which the controller is currently commanding it to move to as opposed to the actual position which is the real position value that is being read by the controller via the position encoder sensor. Note that inadvertently position error is therefore command position - actual position
CommandVelocity
The command velocity is the velocity at which the controller is commanding it to move.
ConnectedMotorType
The motorType is the defined type of the motor that has been set by the controller. If this value is not corresponding to that type that the user is expecting, the refer to the PCSuite to correctly configure the type.
ErrorCode
(Inherited from INode.)

HomingStatus
The home status indicates the whether the homing status is in idle, in error or in motion
MotionStatus
The motion status indicates whether there is a the motor is currently in, either idle, in error or in motion
ServoOnStatus
The ServoOn status indicates whether the motor coils are currently being energized
SmoothFactor
The smooth factor is the time at which the acceleration or emergency deceleration will take to reach it target value
TargetAcceleration
The acceleration that the profiler will move up to during any motion command
TargetDeceleration
The emergency deceleration that the profiler will move up to during any motion command
TargetEmergencyDeceleration
The emergency deceleration that the profiler will use when a stop command of emergency type is executed
TargetVelocity
The velocity that the profiler will move up to during either MoveRelative(Int32), MoveAbsolute(Int32) or Jog(Int32)
